-Changed zip structure
-Added Nodes to:
-RCS Ball
-RCS Tanks
-Wings (@ioresult)
-Cargo Bays

-Space Shuttle wings + flaps
-Octostrut
-Mk1 Lander can
-Truss adapter
-Radial parachutes

-Added a few more parts
-Added patch for RV-105 v2 introduced in 1.7.0, which needed different numbers from the first version

0.5 - 6/5/2019

-BG: Added nodes to robotic hinges
-KRE: Added nodes to hex leg mount

0.6 - 6/10/2019 | Hotfix
-Fixed the issue with nodes not moving correctly when changing Target Angle on Breaking Ground hinges in the VAB/SPH

0.7 - 5/9/2022
-Added nodes for new parts in 1.12
-Removed top node for RCS block due to the part variants having 5-way. This line is just commented out, uncomment to get the node back
-Reworked the Engine plate and tube patches to a much shorter wildcard based method

0.8 - 11/19/2023 | Last update by @Geonovast
-Added version file
-Fixed spacing issues with light attachRules
-Added new node for shielded docking port
-Added node for sas controller (MoarActionables)
